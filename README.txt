RepeatCounter-
User enters a 'keyword', which is a one-word string. Followed by a wordbank. After clicking the submit button:
The RepeatCounter contains the function countRepeats() which will count all instances of the keyword found within the wordbank.
The User is then brought to a 'results' page which displays the results. There is a 'Home' button linking back to the Home page.
